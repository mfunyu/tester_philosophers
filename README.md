# tester_philosophers

## installation
```
git clone https://github.com/mfunyu/tester_philosophers.git
```
#### directory tree
```
── philosophers
   ├── Makefile
   ├── includes
   ├── srcs
   └── tester_philosophers
```

## usage
### run all tests
```
./tester_philosophers/test.sh [help]
```
### run advanced test
```
./tester_philosophers/run.sh
```
